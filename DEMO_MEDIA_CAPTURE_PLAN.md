## 60–90 Second Trailer Structure

### Overview

A single-take or lightly edited browser recording showing the game's full identity in under 90 seconds. No voiceover required — let gameplay and music carry it.

---

### Segment Breakdown

**0–5 sec · Logo / Title**
- Start from main menu cold load
- Show title: `PHENIX: NULL EDEN`
- Let the menu music hit and the SYSTEM FEED begin scrolling
- Hold on the title for 3–4 seconds

**5–15 sec · Character Select / Survivors**
- Navigate to CHARACTER SELECT
- Pan over the 3 characters: Cyber Skeleton Warrior · Neon Taekwondo Girl · Cyber Arm Hero
- Optionally highlight element/class labels
- Select a character and enter Endless Mode

**15–30 sec · Normal Endless Combat**
- Show wave 1–2 combat: movement, auto-fire, dash
- Kill combo building on screen
- Power Matrix visible; Data-Core drop and return mechanic
- Use one Ultimate or Pulse Shield to show ability variety
- Include an enemy wave surge or Grid Cache event if possible

**30–45 sec · Relics / Upgrades / Progression**
- Either: capture the Upgrades menu (Grid Credits spend)
- Or: capture a Relic pickup mid-run (relic appears on screen)
- Show the RELICS screen briefly: 3–5 relics visible with descriptions
- Convey: "there is a build layer here"

**45–65 sec · NULL BREACH ARENA**
- Fast-forward (or cut) to Endless ~5:00
- Show the ARENA BREACH notification and EDEN CORE warning
- Show boss gauntlet: 2–3 boss encounters in the sealed arena
- End with boss death and arena clear

**65–80 sec · EDEN CORE / Boss Echo Archive / Eden Memory / Chaos Laws**
- Return to main menu post-run
- Open ACHIEVEMENTS overlay
- Show: BOSS ECHO ARCHIVE (boss recorded), EDEN MEMORY % ticking up, SYSTEM LOGS (fragment unlocking), CHAOS LAWS PREVIEW
- EDEN CORE transmission appearing adds narrative atmosphere

**80–90 sec · End Card**
- Fade to dark / hold on main menu title screen
- Text overlay:

```
▶ Play free in browser

PHENIX: NULL EDEN

InkSpireM Visuals
mariastergiou1985-cpu.github.io/phenix_survivors_game
```

### Trailer Capture Notes

- **Record at 1080p** (1920×1080) if possible — itch.io cover video displays at this resolution
- **Browser: Chrome** — most reliable for recording quality and frame rate
- **Recording tool options:** OBS Studio (free, high quality), ShareX (Windows, free), browser built-in recording, or Windows Game Bar (Win+G)
- **Frame rate:** 30fps minimum, 60fps preferred
- **Audio:** Keep game music ON — the PHENIX OVERDRIVE theme is a strong trailer track
- Hide browser address bar and any debug panels before recording
- Do NOT show the DevTools console during recording
- Avoid showing any error overlays or toast notifications unrelated to gameplay
- If editing: use cuts, not fades between segments — the pacing is fast
