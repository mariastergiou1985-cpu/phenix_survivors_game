# PHENIX: NULL EDEN — Οδηγός Διανομής (στα ελληνικά, για το αρχείο σου)
*Ενημερώθηκε: 12 Ιουλίου 2026*

## Το βασικό που πρέπει να θυμάσαι
Το παιχνίδι ζει ΜΙΑ φορά, στο GitHub. Κάθε `push` που κάνεις ενημερώνει ΑΥΤΟΜΑΤΑ:
1. το **web** (GitHub Pages),
2. το **.exe** για υπολογιστή (είναι «παράθυρο» που δείχνει το live παιχνίδι),
3. το μελλοντικό **Android app** (ίδια λογική, TWA).
Δεν ξαναχτίζεις τίποτα για διορθώσεις στο παιχνίδι. Push = ενημερώθηκαν όλα.

## 1) WEB (έτοιμο, λειτουργεί τώρα)
- Διεύθυνση: https://mariastergiou1985-cpu.github.io/phenix_survivors_game/
- Κινητό: άνοιξέ το στο Chrome → μενού ⋮ → «Προσθήκη στην αρχική οθόνη» →
  τρέχει fullscreen σαν κανονική εφαρμογή (είναι πλήρες PWA).

## 2) EXE ΓΙΑ WINDOWS (έτοιμο, το χτίζει το GitHub μόνο του)
- Κάθε push που αγγίζει τον φάκελο `electron/` (ή χειροκίνητα: Actions → Build
  Windows EXE → Run workflow) το GitHub χτίζει το exe σε ~5 λεπτά.
- Πού το βρίσκεις: σελίδα repo → καρτέλα **Actions** → κλικ στο πιο πρόσφατο run →
  ενότητα **Artifacts** κάτω → **PHENIX-NULL-EDEN-win64.zip** → κατέβασμα →
  Εξαγωγή → μέσα το `PHENIX NULL EDEN.exe`.
- Το exe θέλει internet (φορτώνει το live παιχνίδι) και αυτο-ενημερώνεται για πάντα.
- Νέο exe χρειάζεται ΜΟΝΟ για αλλαγές στο «κουτί»: εικονίδιο, παράθυρο, Steam App ID.

## 3) STEAM (όλα έτοιμα εκτός από τον λογαριασμό)
Τι ΕΧΕΙ γίνει ήδη:
- Achievements bridge μέσα στο παιχνίδι (τα 12 achievements καταγράφονται από τώρα —
  όποιος παίζει web ΔΕΝ θα χάσει τίποτα, θα συγχρονιστούν στο Steam αυτόματα).
- Electron shell + αυτόματο χτίσιμο exe.
- Πλήρης οδηγός: `docs/STEAM_SETUP.md` (πίνακας achievements για copy-paste).
Τι ΜΕΝΕΙ (δικά σου βήματα, όποτε θελήσεις):
1. Λογαριασμός Steamworks (partner.steamgames.com) — τέλος $100 (επιστρέφεται με
   πωλήσεις >$1.000). Παίρνεις App ID.
2. Μου λες το App ID → το βάζω στο preload.js (μία γραμμή) → push → νέο exe.
3. Στο Steamworks dashboard περνάς τα 12 achievements από τον πίνακα + εικονίδια 64x64
   (τέχνη σου) + σελίδα καταστήματος (screenshots, περιγραφή) → review της Valve.

## 4) GOOGLE PLAY / ANDROID (ο δρόμος στρωμένος)
Τι ΕΧΕΙ γίνει ήδη:
- Το παιχνίδι είναι πλήρες PWA (manifest + service worker + εικονίδια) — απαραίτητη
  προϋπόθεση για Android TWA. Πλήρης οδηγός: `docs/PLAYSTORE_SETUP.md`.
Τι ΜΕΝΕΙ (όποτε θελήσεις):
1. Λογαριασμός Google Play Console — τέλος $25 εφάπαξ.
2. Το εργαλείο Bubblewrap φτιάχνει το .aab με 2 εντολές (μπορώ να στήσω και γι' αυτό
   GitHub Action να το χτίζει μόνο του, όπως το exe — πες το μου όταν έρθει η ώρα).
3. Ανεβάζεις ένα αρχείο assetlinks.json στο repo (σου το δίνει το εργαλείο) + το .aab
   στο Play Console + σελίδα καταστήματος → review της Google.
- Και εδώ: μετά τη δημοσίευση, κάθε push σου ενημερώνει το app αυτόματα.
