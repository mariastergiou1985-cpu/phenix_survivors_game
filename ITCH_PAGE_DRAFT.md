## Description

> **NULL EDEN was not built to save humanity. It was built to preserve the last usable signal.**

PHENIX: NULL EDEN is a browser-based cyber-survival roguelite where you play as PHENIX — a digital signal sent into a collapsing Grid to recover what was lost, face what corrupted it, and echo what came before.

Fight endless waves of cyber-threats. Protect Power Matrices. Recover Data-Cores. Survive escalating boss encounters. Unlock permanent progression across every run.

Every run extends your EDEN MEMORY. Every boss you defeat leaves an echo. Every echo makes you stronger — but the Grid grows darker with each cycle.

**This is an active development build. The core loop is fully playable.**

## Key Features

### Playable Characters
- **Cyber Skeleton Warrior** — Tank/Survival · Electric element · Storm Saber
- **Neon Taekwondo Girl** — Agility/Strike · unique combo-driven ultimate
- **Cyber Arm Hero** — Power/Range · heavy-hitting arm cannon

Each character has distinct stats, auto-fire patterns, a unique Special Move, and element-specific synergies.

### Core Survival Loop
- Protect Power Matrices from enemy waves
- Recover and return Data-Cores before they're stolen
- Prevent Network Overload from reaching 100%
- Survive escalating enemy density across a 32-minute run (or go Endless)

### Boss Encounters
- **AI Overload Titan** — primary boss with phased escalation
- **Matrix Annihilator** — mid-game threat that targets and drains your Matrices directly
- **Bloodfang Packmaster** — fast cyber-beast pack leader at the 10:00 mark, hunts in a pack of Razorhounds
- **NULL BREACH ARENA** — a late-game boss gauntlet triggered in Endless Mode; multiple consecutive boss encounters in a sealed arena

### RELICS System
Equip powerful relic artifacts that alter your build and survival strategy. Relics are found mid-run and create unique synergies per character.

### EDEN CORE — Narrative Transmission System
A mysterious AI presence embedded in the Grid. EDEN CORE speaks to you through in-run transmissions — cryptic messages, system warnings, and fragments of lore that deepen with each run. An evolving portrait reacts to in-run events.

### Permanent Progression
- **EDEN MEMORY** — a persistent % that grows across all runs; unlocks new content, archive entries, and passive bonuses
- **Boss Echo Archive** — defeated bosses leave permanent echoes; echoes apply passive stat bonuses across all future runs
- **Eden Memory Milestones** — named progression thresholds unlock archive entries and system lore
- **Chaos Laws Preview** — a glimpse of the Grid's deep instability rules; expands with progression
- **System Logs / Lore Archive** — hidden lore fragments unlock as your EDEN MEMORY grows; a record of NULL EDEN's true history
- **Grid Credits** — earned per run, spent on permanent character upgrades

### Platform & Controls
- Runs in any modern browser — no install required
- Keyboard · Controller (Xbox / PS / PC) · Mobile touch (landscape)
- Auto-fire — no mouse required during gameplay
