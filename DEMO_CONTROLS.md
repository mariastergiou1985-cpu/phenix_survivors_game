# DEMO CONTROLS — PHENIX: NULL EDEN

---

## Keyboard Controls

| Key | Action |
|---|---|
| `W` `A` `S` `D` | Move (or Arrow Keys) |
| `SHIFT` | Dash |
| `SPACE` | Ultimate — character-specific special move |
| `Q` | Pulse Shield |
| `E` | EMP Cloud (stun burst) |
| `T` | Toggle Aim Assist on/off |
| `M` | Mute / Unmute music |
| `ESC` | Back / Pause / Close overlay |
| `Enter` | Confirm / Select |
| `↑` `↓` | Navigate menus |

Auto-fire is **automatic** — no mouse click required during gameplay.

---

## Controller Support

Xbox, PlayStation, and generic PC controllers are supported.

| Button | Action |
|---|---|
| Left Stick / D-Pad | Move |
| A / Cross | Confirm / Select |
| B / Circle | Back / Cancel |
| Left Bumper (LB/L1) | Dash |
| Right Bumper (RB/R1) | Ultimate |
| X / Square | Pulse Shield |
| Y / Triangle | EMP Cloud |

> **Known caveat:** Controller support is present for gameplay and core navigation. Some upgrade menu and sub-menu navigation sequences may still be refined in upcoming builds. Keyboard fallback is always available.

---

## Mobile Touch Controls (Landscape)

Touch controls are active automatically in landscape orientation on mobile browsers.

| Control | Action |
|---|---|
| Left virtual joystick | Move |
| Action buttons (right side) | Dash · Ultimate · Shield · EMP |
| Tap menu items | Navigate / Select |

**Recommended:** Landscape orientation only. Portrait mode is not supported.
**Tested:** Chrome (Android). iOS Safari should work but is not primary.

---

## Menu Navigation Notes

- Use `↑` `↓` arrow keys or D-Pad to navigate main menu items
- Press `Enter` or `A` (controller) to select
- Press `ESC` or `B` (controller) to go back from any screen
- The **ACHIEVEMENTS**, **RELICS**, **CHARACTER SELECT**, and **UPGRADES** screens are all keyboard and controller navigable
- On mobile, all menu screens are touch-navigable

---

## Aim Assist

Press `T` to toggle aim assist during gameplay. When active, projectiles track toward the nearest enemy within range. When off, directional aim only.

---

## Audio

Press `M` at any time to mute or unmute the game music. Sound effects remain independent of the music toggle.
