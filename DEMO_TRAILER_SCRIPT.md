# PHENIX: NULL EDEN — Demo Trailer Script
## Target Duration: 60–90 seconds

---

## Production Notes

- Record from: `https://mariastergiou1985-cpu.github.io/phenix_survivors_game/`
- Resolution: 1920×1080 preferred / 1280×720 minimum
- Frame rate: 60fps if stable, 30fps fallback
- Browser: Google Chrome, zoom 100%, no DevTools, no console, no extensions visible
- Use landscape orientation
- No taskbar, no desktop chrome visible — use fullscreen browser (F11)
- No black screen — load game fully before recording
- Cut audio cleanly at each scene boundary
- Use game music if capturing with audio

---

## Scene-by-Scene Script

---

### Scene 1 — Title / Main Menu (0–5 sec)

**Visual**: Main menu. Dark atmospheric background. PHENIX: NULL EDEN logo.

**Text overlay (optional)**:
> A dark cyber-survival roguelite

**Capture tip**: Let the ambient background animation play. Show the full title.

---

### Scene 2 — Character Select / Survivors (5–15 sec)

**Visual**: CHARACTER SELECT screen. Multiple playable survivors visible.

**Highlight**:
- Different roles shown (Tank, Ranged, Speed, Endless boss character)
- Oni Cataclysm Protocol if unlocked / visible as locked unlock goal
- Character portraits and names

**Capture tip**: Scroll or hover across characters. Show variety.

---

### Scene 3 — Endless Combat (15–30 sec)

**Visual**: Endless mode in-game action.

**Highlight**:
- Enemy swarms
- Player weapons firing
- Active relic HUD (glowing slots in corner)
- Floating damage numbers
- Neon visual effects

**Capture tip**: Play for 1–2 minutes before recording to get active relics equipped. Show fluid movement + weapon fire. Bosses or elites visible if possible.

---

### Scene 4 — Relics / Progression (30–45 sec)

**Visual**: RELICS screen — show ALL tab and ARENA tab.

**Highlight**:
- Relic categories: UNIVERSAL / BOSS / CHARACTER / ARENA
- Arena-specific relics: Breach Crown, Second Signal Debt, Elite Signal Core
- Relic descriptions showing passive bonuses

**Capture tip**: Switch between tabs on screen. Pause on ARENA tab.

---

### Scene 5 — NULL BREACH ARENA / Boss Pressure (45–65 sec)

**Visual**: NULL BREACH ARENA trigger + boss gauntlet.

**Highlight**:
- ARENA WARNING message if capturable
- Mini-boss spawns (Titan / Bloodfang / Annihilator)
- Major boss (Cyber Serpent / Dragon / Double Demons)
- 2 aircraft active
- Player under heavy pressure
- HUD boss HP bar updating

**Capture tip**: Reach 5-minute mark in Endless mode. The arena triggers automatically. Oni's quad laser is ideal here for visual impact.

---

### Scene 6 — EDEN CORE / Narrative Systems (65–80 sec)

**Visual**: ACHIEVEMENTS screen showing Boss Echo Archive, Eden Memory, System Logs.

**Highlight**:
- Boss Echo Archive: killed boss entries with lore snippets
- Eden Memory milestones timeline
- System Logs: EDEN CORE transmissions
- Chaos Laws Preview tab (if visible)

**Capture tip**: Open ACHIEVEMENTS after a completed run. Shows progression depth.

---

### Scene 7 — End Card (80–90 sec)

**Visual**: Return to main menu or black fade.

**Text overlays**:
> Play free in browser

> PHENIX: NULL EDEN

> InkSpireM Visuals

**Credits roll (optional)**:
> Game / direction / visuals / implementation: Maria Papananou — InkSpireM Visuals
> Music / orchestration: Georgios Litsas
